# Dice Rolling Simulation

A simple Python program that simulates rolling a pair of dice

## Features

-   Simulates any number six‑sided dice
-   Uses Python's `random` module
-   User can roll repeatedly

## Requirements

-   Python 3.x

## How to Run

``` bash
python die_game.py
```

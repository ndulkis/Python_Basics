# Python Basics Projects

This repository contains a collection of small beginner-friendly Python
projects created while learning fundamental programming skills. Each
project focuses on core concepts such as variables, loops, conditionals,
functions, and simple logic building.

## Projects Included

-   **Dice Rolling Game** 
-   **Number Guessing Game** 
-   **Rock, Paper, Scissors** 
-   **QR Code Generator** 
-   **Currency Converter** 
-   **Quiz Game** 
-   **Tic Tac Toe Game** 
-   **Todo List Application** 
-   **Simple Text Editor** 
-   **Pig Dice Game** 
-   **Cows and Bulls Game** 
-   **Password Strength Checker** 
-   **Password Generator**
-   **Word Guessing Game** 
-   **Slot Machine Game**
-   **ATM Simulation** 

## Purpose

These projects were created to practice:

-   Basic Python syntax\
-   Input/output\
-   Loops and conditionals\
-   Functions\
-   Simple game logic\
-   Working with libraries\
-   Building small programs from scratch

## How to Run a Project

Navigate into the project folder and run:

``` bash
python main.py
```

## Structure

Each project has its own folder with its own code and files.

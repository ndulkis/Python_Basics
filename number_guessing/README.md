# Number Guessing Game

A simple Python program that runs a number guessing game

## Features

-   Simple number guessing game in console
-   Uses Python's `random` module
-   User can play repeatedly

## Requirements

-   Python 3.x

## How to Run

``` bash
python main.py
```
